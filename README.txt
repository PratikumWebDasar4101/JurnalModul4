Buatlah sebuah aplikasi yang mampu menyimpan data user. Data user terdiri dari gambar dan hobi. Hobi satu orang user dapat memiliki lebih dari satu hobi. Dengan fitur :
-Delete Hobi
-Upload Foto

Ketentuan Jurnal :
* Menggunakan Array
* CheckBox
* Setelah User mengisi data maka semua data akan di tampilkan di halaman yang lain
